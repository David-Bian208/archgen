#### 第三步：临床推理与鉴别

**推理逻辑（核心）：**
```python
def clinical_reasoning(user_input, hypotheses):
    # 1. 为每个假设寻找支持/反对证据
    for hypothesis in hypotheses:
        supporting = find_supporting_evidence(user_input, hypothesis)
        opposing = find_opposing_evidence(user_input, hypothesis)
        confidence = calculate_confidence(supporting, opposing)
        if confidence < threshold:
            exclude_hypothesis(hypothesis)
    
    # 2. 保留证据最强的 1-2 个假设
    return select_best_hypothesis(hypotheses)
```

**推理示例（薯片盒案例）：**
```
假设 H1：心理理论发展延迟
- 支持：她以为妈妈看到的是糖（无法推断他人信念）
- 反对：无
- 置信度：高 ✅

假设 H2：工作记忆短板
- 支持：需要同时记住 3 个信息
- 反对：她能准确说出盒子里是糖果
- 置信度：中 ⚠️

结论：H1 证据最强
```

#### 2. RDI（人际关系发展干预）

**核心理念：**
- 动态智能 vs 静态技能
- 共同调控（Co-regulation）
- 引导式参与（Guided Participation）
- 社会参照（Social Referencing）

**核心概念：**
```
动态智能 = 在不确定环境中灵活适应的能力
共同调控 = 成人调整自己的行为以支持孩子
社会参照 = 在不确定时看向他人获取信息
```

**可整合内容：**
- ✅ "我们开始玩了吗"信号练习（已在系统中）
- ✅ 视频回放技术（录像分析）
- ✅ 最小化提示原则

**整合示例：**

```markdown
- 等待孩子主动发起
```
