### 3.2 修改 API 地址

**重要：** 需要将 API 地址改为公网地址

**文件：** `frontend/utils/api.js`

**修改前：**

## 🔧 常见问题

### Q1: 提示"不在小程序后台设置的合法域名列表中"

**解决方案：**
1. 开发工具 → 右上角"详情"
2. "本地设置"
3. 勾选"不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书"
4. 重新编译

**正式发布前需要：**
1. 小程序后台 → 开发 → 开发管理 → 开发设置
2. 服务器域名 → request 合法域名
3. 添加：`https://your-domain.com`

---

### Q2: 上传失败

**可能原因：**
- 项目目录选择错误
- `project.config.json` 配置错误
- AppID 填写错误

**解决方法：**
1. 检查 AppID：`wxe260e43ba0375cde`
2. 检查项目目录是否正确
3. 重新导入项目

---

### Q3: 手机预览时无法连接 API

**原因：** 手机无法访问 localhost

**解决：**
1. 修改 `api.js` 中的 BASE_URL
2. 使用公网 IP 或域名
3. 重新编译上传

---

### Q4: 体验版二维码过期

**原因：** 体验版二维码有效期 7 天

**解决：**
1. 重新上传代码
2. 生成新的二维码
