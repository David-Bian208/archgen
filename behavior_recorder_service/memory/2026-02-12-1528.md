## 📁 第一步：创建解码脚本

在你的电脑上创建一个新文件 **`decode.ps1`**，复制粘贴以下全部内容：

```powershell
# 自闭症AI助教项目 - 一键解码脚本 (PowerShell)
# 使用方法：
# 1. 将战舰发你的所有Base64数据保存为 autism_base64.txt（一个文件）
# 2. 右键此脚本 → "使用PowerShell运行"
# 3. 项目将解压到当前目录的 autism_assistant 文件夹

Write-Host "🔧 自闭症AI助教项目解码工具" -ForegroundColor Cyan
Write-Host "========================================"

# 检查Base64文件
$base64File = "autism_base64.txt"
if (-not (Test-Path $base64File)) {
    Write-Host "❌ 找不到Base64文件: $base64File" -ForegroundColor Red
    Write-Host ""
    Write-Host "📝 请按照以下步骤操作：" -ForegroundColor Cyan
    Write-Host "1. 创建一个新文本文件，命名为 autism_base64.txt" -ForegroundColor White
    Write-Host "2. 将战舰发你的所有Base64数据复制粘贴到这个文件" -ForegroundColor White
    Write-Host "3. 确保没有多余的空格或换行（只包含Base64字符）" -ForegroundColor White
    Write-Host "4. 将此脚本和 autism_base64.txt 放在同一文件夹" -ForegroundColor White
    Write-Host "5. 再次运行此脚本" -ForegroundColor White
    Write-Host ""
    pause
    exit 1
}

Write-Host "✅ 找到Base64文件: $base64File" -ForegroundColor Green

# 读取Base64数据
Write-Host "📖 读取Base64数据..." -ForegroundColor Cyan
try {
    $base64Data = Get-Content $base64File -Raw
    # 移除所有空白字符（空格、换行、制表符）
    $base64Data = $base64Data -replace '\s', ''
} catch {
    Write-Host "❌ 读取Base64文件失败：" -ForegroundColor Red
    Write-Host $_.Exception.Message -ForegroundColor Yellow
    pause
    exit 1
}

Write-Host "📊 Base64数据长度: $($base64Data.Length) 字符" -ForegroundColor Cyan

# 解码Base64
Write-Host "🔧 解码Base64..." -ForegroundColor Cyan
try {
    $bytes = [Convert]::FromBase64String($base64Data)
} catch {
    Write-Host "❌ Base64解码失败：" -ForegroundColor Red
    Write-Host $_.Exception.Message -ForegroundColor Yellow
    Write-Host "可能原因：" -ForegroundColor Yellow
    Write-Host "1. Base64数据不完整" -ForegroundColor White
    Write-Host "2. 包含非法字符" -ForegroundColor White
    Write-Host "3. 有空格或换行未清除" -ForegroundColor White
    Write-Host ""
    Write-Host "请检查 autism_base64.txt 文件内容是否正确。" -ForegroundColor Cyan
    pause
    exit 1
}

Write-Host "✅ Base64解码成功，得到 $($bytes.Length) 字节数据" -ForegroundColor Green

# 保存为ZIP文件
$zipFile = "autism_assistant.zip"
Write-Host "💾 保存为ZIP文件: $zipFile..." -ForegroundColor Cyan
try {
    [IO.File]::WriteAllBytes($zipFile, $bytes)
} catch {
    Write-Host "❌ 保存ZIP文件失败：" -ForegroundColor Red
    Write-Host $_.Exception.Message -ForegroundColor Yellow
    pause
    exit 1
}

Write-Host "✅ ZIP文件保存成功" -ForegroundColor Green

# 解压ZIP文件
Write-Host "📂 解压项目文件..." -ForegroundColor Cyan
try {
    # 方法1: 使用System.IO.Compression.ZipFile（需要.NET 4.5+）
    Add-Type -AssemblyName System.IO.Compression.FileSystem
    [System.IO.Compression.ZipFile]::ExtractToDirectory($zipFile, ".")
} catch {
    Write-Host "⚠️  System.IO.Compression.ZipFile解压失败，尝试Expand-Archive..." -ForegroundColor Yellow
    try {
        # 方法2: 使用Expand-Archive（PowerShell 5.0+）
        Expand-Archive -Path $zipFile -DestinationPath "." -Force
    } catch {
        Write-Host "❌ 解压失败：" -ForegroundColor Red
        Write-Host $_.Exception.Message -ForegroundColor Yellow
        Write-Host ""
        Write-Host "🔧 备选方案：" -ForegroundColor Cyan
        Write-Host "1. 手动解压 $zipFile 文件" -ForegroundColor White
        Write-Host "2. 使用WinRAR或7-Zip解压" -ForegroundColor White
        pause
        exit 1
    }
}

# 清理临时文件
Write-Host "🧹 清理临时文件..." -ForegroundColor Cyan
Remove-Item $zipFile -ErrorAction SilentlyContinue
Remove-Item $base64File -ErrorAction SilentlyContinue

Write-Host ""
Write-Host "🎉 项目解压完成！" -ForegroundColor Green
Write-Host "========================================"
Write-Host "项目位置: $(Get-Location)\autism_assistant" -ForegroundColor Yellow
Write-Host ""
Write-Host "🚀 接下来：" -ForegroundColor Cyan
Write-Host "1. 打开命令行 (cmd 或 PowerShell)" -ForegroundColor White
Write-Host "2. 进入项目文件夹: cd autism_assistant" -ForegroundColor White
Write-Host "3. 安装依赖: npm install" -ForegroundColor White
Write-Host "4. 启动应用: npx expo start --web" -ForegroundColor White
Write-Host "5. 浏览器打开 http://localhost:19006" -ForegroundColor White
Write-Host ""
Write-Host "📱 项目包含一键启动脚本 start.bat" -ForegroundColor Cyan
Write-Host ""
pause
```

保存方法：
1. 右键桌面 → 新建文本文档
2. 重命名为 `decode.ps1`（注意扩展名必须是 `.ps1`）
3. 右键 → 编辑，粘贴上面的全部内容，保存
