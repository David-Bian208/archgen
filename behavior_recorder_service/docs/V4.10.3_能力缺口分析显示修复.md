# V4.10.3 能力缺口分析显示修复

## 问题描述

用户反馈：能力缺口分析的编号在视觉上被显示为 4 和 5，而不是 1 和 2。

**实际 LLM 输出：**
```
3. 能力缺口分析：

4. 恰当的社交沟通技能：'发出奇怪声音'→...
5. 社交信号监测：'老师看她时她就笑'→...
```

**期望输出：**
```
3. 能力缺口分析：
   • 恰当的社交沟通技能：'发出奇怪声音'→...
   • 社交信号监测：'老师看她时她就笑'→...
```

## 根本原因

前端 Markdown 解析器不支持嵌套有序列表。当 LLM 生成：
```
3. 能力缺口分析：
   1. 缺口 1
   2. 缺口 2
```

前端解析为连续编号（因为浏览器继续计数），导致显示为 4. 5.

## 修复方案

### 1. 后端修复：修改 LLM Prompt + 后处理

**文件：** `app/agents/insight_analyzer.py`

**修改内容：**
1. **Prompt 修改**：将能力缺口分析的列表格式从有序列表（1. 2. 3.）改为无序列表（•）
2. **后处理逻辑**：新增 `_fix_capability_gap_format()` 方法，强制转换有序列表为无序列表

**为什么需要后处理？**
- LLM 经常忽略 Prompt 要求，仍然生成 1. 2. 3. 有序列表
- 后处理是"防御性编程"，确保格式正确

**修改前：**
```
3. **能力缺口分析**：
   1. **[缺口名称]**：'[行为原文]'→[能力归因]
   2. **[缺口名称]**：'[行为原文]'→[能力归因]
```

**修改后：**
```
3. **能力缺口分析**：
   • **[缺口名称]**：'[行为原文]'→[能力归因]
   • **[缺口名称]**：'[行为原文]'→[能力归因]
```

**后处理代码：**
```python
def _fix_capability_gap_format(self, result: dict) -> dict:
    """强制转换能力缺口分析的有序列表为无序列表"""
    clinical_diff = result.get("clinical_differential", "")
    if "能力缺口分析" not in clinical_diff:
        return result
    
    # 分割文本，找到能力缺口分析部分
    parts = clinical_diff.split("能力缺口分析")
    prefix = parts[0] + "能力缺口分析"
    suffix = parts[1]
    
    # 将有序列表（数字 + 点）转换为无序列表（•）
    import re
    suffix_fixed = re.sub(r'\n\s*(\d+)\.(\s*\*\*)', r'\n   •\2', suffix)
    
    result["clinical_differential"] = prefix + suffix_fixed
    return result
```

### 2. 前端修复：支持无序列表渲染

**文件：** `frontend/App.vue`

**修改内容：**
1. `formatMarkdown()` 函数新增无序列表匹配逻辑
2. 支持 `•`、`-`、`*` 三种无序列表符号
3. 添加 CSS 样式支持 `<ul>` 和嵌套列表

**新增 CSS：**
```css
.clinical-text ul {
  margin: 12px 0;
  padding-left: 24px;
  list-style-type: disc; /* 实心圆点 */
}

.clinical-text li > ul {
  margin-top: 4px;
  padding-left: 20px;
  list-style-type: circle; /* 空心圆 */
}
```

## 测试验证

### 测试案例 #4（寻求关注）

**输入：**
```
我女儿在课堂上总是突然发出奇怪的声音，老师看她时她就笑，不看她时就继续发出声音，即使被批评也停不下来。
```

**期望输出 - 能力缺口分析部分：**
```
3. **能力缺口分析**：
   • **社交沟通技能不足**：'发出奇怪声音'→表明在如何用恰当方式（如举手、安静等待）获取成人注意力上存在困难
   • **社交信号监测弱**：'老师看她时她就笑'→表明能监测到老师是否在关注自己，但缺乏在获取关注后如何维持恰当互动的技能
   • **自我调节能力不足**：'即使被批评也停不下来'→表明在兴奋/冲动时难以控制行为强度
```

**视觉渲染效果：**
- 3. 能力缺口分析：（有序列表第 3 项）
  - • 社交沟通技能不足：...（无序列表子项）
  - • 社交信号监测弱：...（无序列表子项）
  - • 自我调节能力不足：...（无序列表子项）
