## 问题 2：能力缺口分析还是太叙述 ❌

### 用户反馈
> "能力缺口分析，少一些叙述，说一下原因，比如为什么会这样？视听动问题还是什么"

### 当前输出（问题示例）
```
2. 能力缺口分析：
   • 观点采择能力弱：'她以为我看到盒子里还是糖'→表明难以理解"我知道的"和"别人知道的"可以不同...
   • 认知灵活性不足：'还是以为妈妈是在用电脑，爸爸还是在用手机'→表明在情境（位置）发生变化后，难以更新...
```

**问题：**
- ❌ 还是在复述行为（"她以为..."）
- ❌ 没有解释"为什么"会有这个困难
- ❌ 没有提到具体发展原因（视听动？工作记忆？神经发育？）

### 期望输出（深度归因）
```
2. 能力缺口分析：
   
   • **观点采择能力弱**：这可能与孩子的**心理理论（Theory of Mind）发展节奏**有关。大脑中负责理解他人心理状态的区域（如颞顶联合区）在 4-6 岁还在快速发育中，有些孩子会稍晚一些。这不是"笨"或"不听话"，而是神经发育的时间表不同。可以通过"错误信念"游戏（如薯片盒子装糖果）来温和刺激这个能力的发展。
   
   • **认知灵活性不足**：这涉及到**执行功能中的工作记忆和认知切换能力**。当孩子需要同时记住"刚才的位置"和"现在的位置"，并快速切换时，大脑的前额叶皮层需要处理较大负荷。有些孩子的工作记忆容量发展稍慢，或者在兴奋情境下（如游戏）更难分配注意力。可以通过慢节奏的"换位置猜物品"游戏来练习，逐步提升认知灵活性。
```

**关键差异：**
- ✅ 解释生理/神经原因（"颞顶联合区发育中"、"前额叶皮层负荷"）
- ✅ 提到具体能力（"心理理论"、"工作记忆"、"认知切换"、"执行功能"）
- ✅ 正常化（"神经发育的时间表不同"、"有些孩子会稍晚"）
- ✅ 指出练习方向

### 修复方案

**文件：** `app/agents/insight_analyzer.py`

**修改位置：** `ANALYSIS_SYSTEM_PROMPT` 中的"能力缺口分析"部分

**当前提示词：**
```python
3. **能力缺口分析**（从现象到归因）
- 格式：每个缺口必须包含三层结构
  1. **行为表现**：简短引用孩子的具体行为（用引号标注）
  2. **发展解读**：解释这个表现反映了什么发展阶段特点，引用发展心理学理论...
  3. **正常化 + 方向**：说明这是发展过程中的常见现象，指出能力发展方向
```

**修改为：**
```python
3. **能力缺口分析**（从现象到生理/神经归因）
- 格式：每个缺口必须包含三层结构
  1. **行为表现**：极简，5-10 字以内（如"'以为妈妈看到糖'"）
  2. **神经/发展归因**：解释**为什么**会有这个表现，必须提到以下至少一项：
     - **脑区发育**：如"颞顶联合区（负责心理理论）还在发育中"、"前额叶皮层（负责执行功能）未成熟"
     - **神经机制**：如"神经连接还在建立中"、"突触修剪未完成"
     - **认知机制**：如"工作记忆容量有限"、"注意力分配能力不足"、"认知切换需要时间"
     - **发展节奏**：如"有些孩子发展稍晚"、"神经发育时间表不同"
  3. **正常化 + 方向**：说明不是缺陷，指出练习方向

- **示例**：
  • **观点采择能力弱**：这与**心理理论（Theory of Mind）发展节奏**有关。大脑中负责理解他人心理状态的区域（颞顶联合区）在 4-6 岁还在快速发育中，有些孩子会稍晚一些。这不是"笨"或"不听话"，而是神经发育的时间表不同。可以通过"错误信念"游戏来温和刺激这个能力的发展。
  
  • **认知灵活性不足**：这涉及到**执行功能中的工作记忆和认知切换能力**。当孩子需要同时记住两套信息并快速切换时，大脑的前额叶皮层需要处理较大负荷。有些孩子的工作记忆容量发展稍慢，或者在兴奋情境下更难分配注意力。可以通过慢节奏的"换位置猜物品"游戏来练习。

- ❌ 避免：
  • 长篇复述行为（超过 15 字）
  • 只说"是什么"不说"为什么"
  • 缺少神经/生理归因
  
- ✅ 提倡：
  • 行为描述极简（5-10 字）
  • 重点解释"为什么"（脑区、神经、认知机制）
  • 正常化（"发育中"、"时间表不同"、"有些孩子会稍晚"）
```
