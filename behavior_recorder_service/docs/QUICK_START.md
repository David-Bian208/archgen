# 行为观察伙伴 - 快速使用指南

**一句话说明：** 这是一款帮助您理解孩子行为的专业工具，打开即用，无需注册。

---

## 🚀 3 分钟上手

### 第 1 步：打开系统
浏览器访问：`http://localhost:3000`

### 第 2 步：描述行为
在输入框中输入您观察到的孩子行为。

**示例：**
```
在幼儿园做操时，小明一直盯着旁边的小朋友看，不跟着做动作。
老师提醒后，他看看老师，又开始看小朋友。
```

**小技巧：**
- ✅ 说清楚时间、地点、具体行为
- ✅ 描述他人反应和孩子回应
- ❌ 不要用专业术语或评价性语言

### 第 3 步：等待分析
点击"发送"按钮，等待 60-90 秒。

### 第 4 步：查看结果
您会看到：
- 📋 **分析报告**：行为功能、核心能力目标
- 📚 **干预建议**：游戏化练习、成功标准、观察任务

### 第 5 步：提交反馈（可选）
滚动到页面底部，点击"提交反馈"，帮助我们改进。

---

## 💡 常见问题

**Q: 分析需要多长时间？**  
A: 60-90 秒。如果超过 2 分钟，请刷新重试。

**Q: 报告准确吗？**  
A: 基于专业理论，但仅供参考，不能替代专业评估。

**Q: 如何保存记录？**  
A: 数据自动保存，点击"历史记录"查看。

**Q: 遇到问题怎么办？**  
A: 检查网络连接，刷新重试，或提交反馈。
