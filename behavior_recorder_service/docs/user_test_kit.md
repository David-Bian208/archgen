## ✅ 必做任务（3 个场景）

### 任务 1：寻求关注行为分析 ⭐

**场景：** 孩子在课堂上故意制造声音吸引老师注意

**操作步骤：**
1. 打开网页
2. 在输入框输入：
   ```
   我儿子在课堂上总是突然发出奇怪的声音，老师看他时他就笑，
   不看她时就继续发出声音，即使被批评也停不下来。
   ```
3. 点击"发送"
4. 等待 AI 生成报告（约 5-10 秒）
5. 仔细阅读报告
6. 在报告末尾评分并提交反馈

**测试重点：**
- [ ] 分析是否符合你的直觉？
- [ ] 建议是否有用？
- [ ] 语言是否好懂？

**预计用时：** 3 分钟

---

### 任务 2：逃避行为分析 ⭐

**场景：** 孩子写作业时找各种借口拖延

**操作步骤：**
1. 点击"开始新的记录"（或刷新页面）
2. 输入：
   ```
   我女儿每天写作业时总是要去喝水、上厕所、削铅笔，
   一会儿说饿了，一会儿说累了，就是不想写作业。
   ```
3. 点击"发送"
4. 等待报告生成
5. 阅读报告并提交反馈

**测试重点：**
- [ ] 分析是否准确？
- [ ] 建议是否可操作？
- [ ] 有没有让你意外的洞察？

**预计用时：** 3 分钟

---

### 任务 3：自由对话测试 ⭐

**场景：** 随便问任何关于孩子行为的问题

**操作步骤：**
1. 点击"开始新的记录"
2. 输入任何你想问的问题，例如：
   - "孩子发脾气怎么办？"
   - "如何培养孩子的专注力？"
   - "孩子在学校打人，应该怎么教育？"
3. 可以和 AI 多轮对话（追问）
4. 最后提交反馈

**测试重点：**
- [ ] AI 能否理解你的问题？
- [ ] 回答是否连贯？
- [ ] 有没有答非所问？

---

## 使用相关

### Q1: 如何使用这个工具？

3. 点击"发送"

5. 阅读报告并提交反馈

### Q5: AI 分析准确吗？
**A:** AI 基于行为分析理论（ABC 理论）生成建议，但：
- ✅ 可以作为参考
- ⚠️ 不能替代专业评估
- ⚠️ 严重问题请咨询专业人士

2. 分析行为功能
3. 生成个性化建议

**A:** 可以！在 AI 回复后继续输入问题，AI 会结合上下文回答。
