## 📊 审核标准

| 等级 | 标记 | 标准 |
|------|------|------|
| **通过** | ✅ | 高频临床表达，有明确文献依据，无歧义 |
| **修改** | ⚠️ | 语义模糊、有更优表达、需调整措辞 |
| **删除** | ❌ | 过于主观、生造词、重复、临床不常见 |
| **待定** | ❓ | 需要更多临床证据或讨论 |
