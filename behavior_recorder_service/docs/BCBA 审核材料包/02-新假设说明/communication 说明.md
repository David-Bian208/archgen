## 一、假设定义

### 1.1 基本信息

| 属性 | 内容 |
|------|------|
| **假设 ID** | communication |
| **名称** | 沟通能力挑战 |
| **描述** | 语言表达或理解能力不足，影响有效沟通 |
| **先验概率** | 0.35 |
| **层级** | 中层归因假设 |
| **父假设** | H_ESCAPE (逃避) |
| **子假设** | - |

### 1.2 临床定义

沟通能力挑战指个体在语言表达或理解方面存在困难，影响有效沟通，表现为：
- 表达困难：无法清楚表达自己的需求、感受或想法
- 理解困难：无法理解他人的语言指令或信息
- 功能性沟通弱：不会用适当方式求助、拒绝或分享
- 语言发育迟缓：语言能力落后于同龄人

## 二、证据规则

### 2.1 支持证据 (10 条)

| 证据 | 行为表现 | 临床意义 |
|------|----------|----------|
| 不会表达 | 无法用语言描述需求 | 表达能力弱 |
| 说不清楚 | 语言表达含糊 | 语言组织弱 |
| 词不达意 | 用词不准确 | 语言组织能力弱 |
| 无法描述感受 | 无法表达情绪 | 情感表达困难 |
| 不会求助 | 不知道如何请求帮助 | 功能性沟通弱 |
| 用行为代替语言 | 用哭闹、打人代替说话 | 沟通能力缺乏 |
| 听不懂指令 | 无法理解语言指令 | 语言理解困难 |
| 不回应 | 对语言刺激无反应 | 语言理解或社交沟通困难 |
| 不说话 | 语言缺乏 | 语言能力严重落后 |
| 语言发育迟缓 | 语言能力落后于年龄 | 发育问题 |

### 2.2 反对证据 (2 条)

| 证据 | 行为表现 | 临床意义 |
|------|----------|----------|
| 能清楚表达 | 能清楚表达需求和感受 | 沟通能力良好 |
| 理解能力正常 | 能理解语言指令 | 语言理解良好 |
