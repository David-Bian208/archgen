## 一、假设定义

### 1.1 基本信息

| 属性 | 内容 |
|------|------|
| **假设 ID** | social_skills |
| **名称** | 社交技能挑战 |
| **描述** | 缺乏同伴交往和社交互动所需技能 |
| **先验概率** | 0.30 |
| **层级** | 中层归因假设 |
| **父假设** | H_ESCAPE (逃避) |
| **子假设** | - |

### 1.2 临床定义

社交技能挑战指个体在同伴交往和社交互动中缺乏必要技能，表现为：
- 交友困难：不知道如何建立和维持友谊
- 社交回避：避免或不参与社交活动
- 社交技能缺乏：不会分享、轮流、合作等基本社交技能
- 冲突解决困难：不知道如何解决社交冲突

## 二、证据规则

### 2.1 支持证据 (9 条)

| 证据 | 行为表现 | 临床意义 |
|------|----------|----------|
| 不会交朋友 | 无法建立友谊 | 社交技能缺乏 |
| 没有朋友 | 社交孤立 | 社交困难的结果 |
| 不合群 | 避免群体活动 | 社交回避 |
| 不知道如何加入 | 不知道如何加入同伴游戏 | 社交技能缺乏 |
| 被排斥 | 被同伴排斥 | 社交困难的结果 |
| 被孤立 | 被同伴孤立 | 社交困难的结果 |
| 不会分享 | 不愿意或不会分享 | 分享技能缺乏 |
| 不会轮流 | 不知道轮流等待 | 轮流技能缺乏 |
| 不会合作 | 无法与同伴合作 | 合作技能缺乏 |

### 2.2 反对证据 (2 条)

| 证据 | 行为表现 | 临床意义 |
|------|----------|----------|
| 有稳定朋友 | 有 1-2 个稳定朋友 | 社交能力存在 |
| 能主动社交 | 能主动发起社交互动 | 社交技能存在 |
