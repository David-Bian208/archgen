## 一、假设定义

### 1.1 基本信息

| 属性 | 内容 |
|------|------|
| **假设 ID** | emotional_regulation |
| **名称** | 情绪调节挑战 |
| **描述** | 情绪反应强度超出情境需要，难以自我调节 |
| **先验概率** | 0.30 |
| **层级** | 中层归因假设 |
| **父假设** | H_ESCAPE (逃避) |
| **子假设** | - |

### 1.2 临床定义

情绪调节挑战指个体在特定情境下情绪反应强度、持续时间或恢复速度超出典型范围，表现为：
- 情绪强度高：反应过于激烈 (如大发脾气)
- 情绪持续时间长：难以从负面情绪中恢复
- 情绪触发阈值低：小事引发大反应
- 自我安抚能力弱：难以自我调节情绪

## 二、证据规则

### 2.1 支持证据 (14 条)

| 证据 | 行为表现 | 临床意义 |
|------|----------|----------|
| 情绪崩溃 | 大哭大闹，无法控制 | 情绪调节完全失效 |
| 控制不住脾气 | 自我报告无法控制 | 自我意识存在但调节困难 |
| 一点就着 | 情绪爆发阈值低 | 情绪敏感 |
| 脾气大 | 经常发脾气 | 情绪调节困难 |
| 易怒 | 容易被激怒 | 情绪敏感 + 调节困难 |
| 执拗 | 固执己见，不妥协 | 情绪灵活性弱 |
| 暴躁 | 情绪爆发快速且强烈 | 情绪调节困难 |
| 情绪化 | 情绪波动大 | 情绪不稳定 |
| 玻璃心 | 情绪敏感，容易受伤 | 情绪敏感 |
| 输不起 | 无法接受失败 | 挫折耐受低 |
| 容易沮丧 | 容易感到挫败 | 挫折耐受低 |
| 叛逆 | 对抗权威 | 情绪调节困难 + 寻求控制 |
| 对着干 | 故意做相反的事 | 情绪调节困难 + 寻求控制 |
| 情绪波动大 | 情绪起伏大 | 情绪不稳定 |

### 2.2 反对证据 (2 条)

| 证据 | 行为表现 | 临床意义 |
|------|----------|----------|
| 情绪稳定 | 情绪反应适中 | 情绪调节良好 |
| 能自我安抚 | 能自己平静下来 | 情绪调节能力存在 |

## 三、鉴别诊断

### 3.1 与相关假设的区别

| 假设 | 核心特征 | 与 emotional_regulation 的区别 |
|------|----------|-------------------------------|
| **impulse_control** | 冲动控制困难 | 侧重行为抑制，emotional_regulation 侧重情绪调节 |
| **H_ESCAPE_DIFFICULTY** | 逃避难度 | 逃避是功能，情绪调节是能力缺口 |
| **H_ATTENTION** | 寻求关注 | 寻求关注是有目的的，情绪爆发可能是无目的的 |

### 3.2 共病情况

| 共病 | 发生率 | 说明 |
|------|--------|------|
| ADHD | 高 | 情绪调节困难是 ADHD 常见伴随症状 |
| DMDD (破坏性情绪失调障碍) | 中 | 慢性易怒 + 情绪爆发 |
| 焦虑障碍 | 中 | 情绪调节困难 + 焦虑 |
| 对立违抗障碍 (ODD) | 中 | 情绪调节困难 + 违抗行为 |

### 5.1 评估方法

| 方法 | 工具 | 说明 |

| 行为观察 | ABC 记录 | 记录情绪爆发的频率、强度、持续时间 |

| 临床访谈 | 结构化访谈 | 了解情绪调节历史 |

### 5.2 干预策略

| 策略 | 方法 | 目标 |
|------|------|------|
| 情绪识别训练 | 教孩子识别情绪信号 | 提升情绪觉察 |
| 自我安抚技巧 | 深呼吸、数数等 | 提升自我调节能力 |
| 认知重构 | 改变对情境的解释 | 降低情绪反应强度 |
| 环境调整 | 减少情绪触发因素 | 降低情绪爆发频率 |
| 挫折耐受训练 | 逐步暴露于挫折情境 | 提升挫折耐受 |
