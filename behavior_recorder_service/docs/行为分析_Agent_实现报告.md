### 实现细节

#### 1. 行为解码 Skills
- **功能：** 从 ABC 描述中提取认知行为单元
- **输入：** ABC 记录（前因 - 行为 - 结果）
- **输出：** 认知行为单元列表（unit_id, type, description）
- **提示词要点：** 剥离情境细节，关注"做了什么"，用中性语言描述

#### 2. 模式提炼 Skills
- **功能：** 将多个认知行为单元聚类为行为模式
- **输入：** 认知行为单元列表
- **输出：** 行为模式列表（pattern_id, pattern_name, description, related_units）
- **提示词要点：** 寻找重复出现的行为特征，用描述性语言命名

#### 3. 多维度假设生成 Skills
- **功能：** 针对行为模式平行列举认知功能解释
- **输入：** 行为模式列表
- **输出：** 假设清单（hypothesis_id, cognitive_domain, description, neural_system, developmental_stage）
- **认知维度框架：** 心理理论、执行功能、视觉空间认知、语言概念理解
- **提示词要点：** 平行列举，不急于下结论，标注关联神经/发展系统

#### 4. 证据权衡 Skills
- **功能：** 评估每个假设的强弱证据
- **输入：** 假设清单 + ABC 观察记录
- **输出：** 证据评估列表（supporting_evidence, contradicting_evidence, weight, confidence）
- **提示词要点：** 寻找支持/反驳证据，评估权重（强/中/弱），给出置信度（0-1）

### 实现细节

**功能：** 整合 Skills 输出，进行阶段定位和整合归因

**处理流程：**
1. Step 1: 行为解码（Skills 1）→ 认知行为单元
2. Step 2: 模式提炼（Skills 2）→ 行为模式
3. Step 3: 多维度假设生成（Skills 3）→ 假设清单
4. Step 4: 证据权衡（Skills 4）→ 证据评估
5. Step 5: Agent 整合 → 发展剖面 + 归因陈述

**输出结构：**
```json
{
  "success": true,
  "developmental_profile": {
    "strengths": ["强项 1", "强项 2"],
    "current_focus": ["焦点 1", "焦点 2"],
    "potential_challenges": ["挑战 1"]
  },
  "attribution_statement": "整合归因陈述（200-300 字）",
  "hypotheses_ranking": [
    {"id": "H1", "rank": 1, "confidence": 0.85}
  ]
}
```
