### 1️⃣ InsightAnalyzer 新增功能判断校验层（P0）

**文件：** `app/agents/insight_analyzer.py`

**位置：** 在 `_add_developmental_perspective()` 方法之前，新增 `_validate_functional_judgment()` 方法

**代码：**

```python
def _validate_functional_judgment(self, result: dict, session_context: dict) -> bool:
    """
    P0 新增：功能判断校验层
    
    校验功能判断与行为描述是否一致，防止"分析正确但归因错误"
    
    Args:
        result: LLM 返回的分析结果
        session_context: 会话上下文（包含原始行为描述）
    
    Returns:
        bool: True 表示校验通过，False 表示需要降级
    """
    behavior = session_context.get("child_behavior", "")
    context = session_context.get("context", "")
    judgment = result.get("functional_judgment", "")
    
    # 定义禁止组合（"开错药"模式）
    forbidden_combinations = [
        # 同伴互动场景 × 寻求关注（应优先社交技能不足）
        ("同伴", "寻求关注"),
        ("玩伴", "寻求关注"),
        ("小朋友", "寻求关注"),
        ("同学", "寻求关注"),
        
        # 能力完备 × 提示依赖（"都会做"排除提示依赖）
        ("都会做", "提示依赖"),
        ("已经会", "提示依赖"),
        ("自己能", "提示依赖"),
        ("不需要帮助", "提示依赖"),
        
        # 无观众 × 寻求关注（一个人玩排除寻求关注）
        ("一个人", "寻求关注"),
        ("自己玩", "寻求关注"),
        ("没人", "寻求关注"),
    ]
    
    check_text = f"{behavior}{context}"
    
    for context_keyword, forbidden_judgment in forbidden_combinations:
        if context_keyword in check_text and forbidden_judgment in judgment:
            logger.warning(
                f"⚠️ P0 功能判断校验失败 | "
                f"行为：{check_text[:50]}... | "
                f"判断：{judgment} | "
                f"禁止组合：'{context_keyword}' × '{forbidden_judgment}' | "
                f"已降级到保守方案"
            )
            return False
    
    logger.debug(f"P0 功能判断校验通过：{judgment[:50]}...")
    return True
```

### 4️⃣ 添加置信度评估（P1）

**文件：** `app/agents/insight_analyzer.py`

**修改：** 在方法末尾添加置信度字段：

```python
# 计算置信度（基于功能判断的明确性）
confidence = self._calculate_confidence(result)
result["confidence"] = confidence
result["confidence_level"] = "high" if confidence > 0.8 else "medium" if confidence > 0.5 else "low"

```

**新增辅助方法：**

```python
def _calculate_confidence(self, result: dict) -> float:
    """
    P1 新增：计算功能判断置信度
    
    基于：
    1. 功能判断的明确性
    2. 临床鉴别思考的详细程度
    3. 能力缺口的具体性
    """
    confidence = 0.5  # 基础置信度
    
    # 1. 功能判断明确性
    judgment = result.get("functional_judgment", "")
    if judgment and len(judgment) > 10 and "不确定" not in judgment:
        confidence += 0.2
    
    # 2. 临床鉴别思考详细程度
    differential = result.get("clinical_differential", "")
    if differential and len(differential) > 100:
        confidence += 0.15
    
    # 3. 能力缺口具体性
    if "capability_hypothesis" in result and len(result["capability_hypothesis"]) > 20:
        confidence += 0.15
    
    return min(confidence, 1.0)
```

## 🧪 测试用例

### 测试 1：同伴互动 × 寻求关注（应触发降级）

**输入：**
```
情境：小明和玩伴一起玩积木
孩子的表现：小明用力推了玩伴，玩伴哭了
周围的反应：老师过来批评小明
```

**预期：**
- 功能判断不应是"寻求关注"
- 置信度应为 low
- 建议降级到保守方案或判断为"社交技能不足"

---

### 测试 2：能力完备 × 提示依赖（应触发降级）

**输入：**
```
情境：做数学题
孩子的表现：其实都会做，但总是叫"妈妈帮我"
周围的反应：妈妈过去帮助
```

**预期：**
- 功能判断不应是"提示依赖"
- 应判断为"寻求关注"
- 置信度 medium 或 high
