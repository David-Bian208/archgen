## 🎯 任务分解

### 任务 1：SECURITY_CHECKLIST.yml（2 小时）

**描述：** 创建机器可读的安全检查清单

**要求：**
1. 至少 10 个检查项
2. 覆盖 OWASP Top 10 核心风险
3. 支持 grep 扫描（正则表达式）
4. 包含严重级别（critical/high/medium/low）

**示例格式：**
- id: no-hardcoded-keys
    description: 禁止硬编码 API 密钥
    pattern: "API_KEY\\s*=\\s*['\"][^'\"]+['\"]"
    severity: critical
    files: ["*.py", "*.js", "*.vue"]
  - id: no-eval
    description: 禁止使用 eval()
    pattern: "\\beval\\s*\\("
    severity: high
    files: ["*.py", "*.js"]
  # ... 至少 10 项
- --

### 任务 2：安全扫描脚本（1 小时）

**描述：** 编写 Python 脚本，基于 SECURITY_CHECKLIST.yml 执行扫描

**要求：**
1. 读取 YAML 配置文件
2. 遍历项目文件（.py/.js/.vue）
3. 执行正则匹配
4. 生成结构化报告（JSON + 人类可读）

**示例输出：**

发现问题：2

[HIGH] no-eval
  文件：app/utils/parser.py:42
  匹配：eval(user_input)
  建议：使用 ast.literal_eval
