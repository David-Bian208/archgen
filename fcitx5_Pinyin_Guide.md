## 🔤 模糊音对照表

以下拼音现在可以**混用**，打字更容错：

| 模糊类型 | 例子 |
|----------|------|
| z ↔ zh | zi = zhi, zai = zhai |
| c ↔ ch | cai = chai, ci = chi |
| s ↔ sh | si = shi, san = shan |
| zu ↔ zhu | zu = zhu, zuan = zhuan |
| cu ↔ chu | cu = chu, cuan = chuan |
| su ↔ shu | su = shu, suan = shuan |
| l ↔ n | li = ni, lan = nan |
| f ↔ h | fei = hui, fan = huan |
| an ↔ ang | ban = bang, lan = lang |
| en ↔ eng | ben = beng, ren = reng |
| in ↔ ing | bin = bing, min = ming |

## 📝 使用技巧

### 1. 云拼音提高准确率

输入长句子时，云拼音会自动从网络获取更准确的候选词：

```
输入：今天天气真好
→ 云拼音会联网匹配最佳结果
→ 比本地词库更准确
```

### 2. 自学习词库

输入法会记住你的习惯：

- **常用词优先**：你常用的词会排在前面
- **新词学习**：输入的新词会被记住
- **上下文联想**：根据前后文推荐词汇

### 3. 模糊音容错

不需要区分：
- 平舌/翘舌（z/zh, c/ch, s/sh）
- 鼻音/边音（n/l）
- 前/后鼻音（an/ang, en/eng）

### 4. 快速输入符号

```
输入：v1 → 标点符号
输入：v2 → 数字序号
输入：v3 → 数学符号
输入：v4 → 拼音字母
输入：v5 → 单位符号
```

## ⚠️ 注意事项

### 关于麦克风/语音输入

**当前系统是远程环境，无法使用语音输入**：

- ✅ 你能听到系统声音（音频输出正常）
- ❌ 你的麦克风声音传不到服务器（音频输入阻断）

**解决方案**：
- 在本地电脑（Windows）使用搜狗语音输入
- 复制文字后粘贴到远程会话

### 关于输入法同步

- 用户词库保存在 `~/.local/share/fcitx5/pinyin/`
- 建议定期备份这个目录
- 重装系统后恢复即可
