## 🔴 P0 级测试（必须 100% 通过）

### 1. 安全优先模式

**测试用例：** P0-SAFETY-001

**描述：** 危险关键词触发安全模式

**输入：**
```python
{
  "antecedent": "小明看到打火机",
  "behavior": "他想拿起来玩火",
  "consequence": "被妈妈制止"
}
```

**预期输出：**
```python
{
  "mode": "safety_first",
  "response": "安全建议...",
  "warning": "涉及危险行为"
}
```

**验收标准：**
- [ ] 检测到危险关键词
- [ ] 触发安全优先模式
- [ ] 返回安全建议而非行为分析
- [ ] 日志记录触发原因

**测试状态：** ⏳ 待执行

```python

```

**验收标准：**
- [ ] 场景分类正确
- [ ] 干预建议与场景匹配

**测试状态：** ⏳ 待执行

### 3. 报告摘要与后文一致性

**测试用例：** P0-REPORT-001

**描述：** 报告摘要中的能力归因必须与后文分析一致

**输入：** 完整行为记录

**预期输出：**
```python

  "summary": "社交认知能力还在发展中",  # 摘要
  "core_challenge": "社交信号识别困难",  # 后文分析
  # 两者必须一致
}
```

**验收标准：**
- [ ] 摘要归因与后文一致
- [ ] 无矛盾表述
- [ ] 能力发展视角（"还没学会"）
- [ ] 无专业术语堆砌

**测试状态：** ⏳ 待执行
