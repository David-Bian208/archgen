## ⚠️ 发现的问题

### 问题 1: 数据收集归类不准确
```
collected_data: {
  "context": "今天在幼儿园...一开始就不想做",  // ❌ 应该是情境
  "behavior": "今天在幼儿园...一开始就不想做",  // ❌ 重复了 context
  "response": "他说不会剪...放下了"  // ✅ 正确
}
```

**原因**: `_update_collected_data()` 只根据当前状态归类，没有用 LLM 提取

**影响**: 分析报告中的 behavior 字段内容与 context 重复

**建议修复**:
```python
def _update_collected_data(self, state, user_input):
    # 使用 LLM 辅助提取，而非简单归类
    if state.current_state == InterviewState.GATHER_CONTEXT:
        extracted = self.llm.extract_context(user_input)
        state.collected_data["context"] = extracted
```

---

### 问题 2: Prompt 未完全遵循"总结先行"
V4.0 Prompt 要求：
> 1. **总结**："基于我们目前的对话，我了解到..."
> 2. **决策**："为了进一步明确，现在最需要厘清的一点是..."
> 3. **提问**：提出唯一的、精准的问题

实际输出：
- 轮次 1: 直接提问，无总结
- 轮次 2: 确认上一轮（"小组环境"），但未总结全部
- 轮次 3: 直接提问，无总结

**原因**: Prompt 中的要求不够强制，LLM 忽略了

**建议修复**: 在 Prompt 中添加示例：
```
【示例回应】
"基于我们的对话，我了解到孩子在幼儿园小组桌上做手工。
为了进一步明确，现在最需要厘清的是他当时的具体行为。
可以请您描述一下，当他说太难时，具体做了什么吗？"
```
