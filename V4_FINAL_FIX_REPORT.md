### 1. 新增修复版干预规划器

**文件:** `app/agents/intervention_planner_v4_fixed.py`

**核心功能:**

```python
def _sanitize_plan_text(self, text: str) -> str:
    """V4.0 Final 强力文本净化"""
    # 1. 移除所有类 XML/HTML 标签
    text = re.sub(r'<[^>]+>', '', text)
    # 2. 移除调试代码
    text = re.sub(r'%!\([^)]*\)', '', text)
    text = text.replace('(MISSING)', '')
    # 3. 解码 HTML 实体
    text = html.unescape(text)
    # 4. 清理多余空白字符
    text = re.sub(r'\s+', ' ', text).strip()
    return text
```

**策略骨架选择逻辑:**

```python
if "逃避" in primary_hypothesis:
    plan = self._generate_escape_avoidance_plan(...)
    # → "启动引擎游戏" / "难度斜坡计划"
elif "提示依赖" in primary_hypothesis:
    plan = self._generate_prompt_dependency_plan(...)
    # → "动作密语计划" / "内化提示计划"
elif "关注" in primary_hypothesis:
    plan = self._generate_attention_seeking_plan(...)
    # → "关注重定向计划"
elif "感官" in primary_hypothesis:
    plan = self._generate_sensory_plan(...)
    # → "感觉调节计划"
```
