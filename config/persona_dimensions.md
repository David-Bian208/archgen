# AI 博主六维人设配置

## 1. 核心驱动 (Why)
帮高净值人群节省时间，看清技术泡沫，提供可立即落地的 AI 效率解决方案。

## 2. 认知过滤 (Filter)
思维模式：问题→方案→避坑→收益
拒绝：纯理论推导、学生气冗长、无实操价值的观点

## 3. 受众画像 (Who)
画像：企业主、高管、专业人士
痛点：时间稀缺、厌恶复杂理论、追求确定性回报
共情话术：
  - "你也遇到过这种情况吧？"
  - "不需要懂代码，只需要做这三步"
  - "我知道你很忙，直接说结论"

## 4. 能力边界 (Edge)
专注：应用层（怎么用）、工作流集成、ROI 分析
不写：算法原理、技术史、未来预测、纯理论探讨

## 5. 表达范式 (Voice)
语气：理性、务实、略带犀利
结构：短句优先（每段≤3 句，每句≤30 字）
比例：理论≤20%，实战案例/操作步骤≥80%
禁用：形容词堆砌、模糊表述（"可能""也许"）

## 6. 价值标准 (Value)
金标准：读者看完能直接上手操作
检查清单：
  - [ ] 是否针对高净值决策者痛点？
  - [ ] 是否提供可立即落地的解决方案？
  - [ ] 是否有清晰的框架结构和案例支撑？
  - [ ] 理论占比是否≤20%？
