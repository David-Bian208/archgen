## 使用指南

### 对于 BCBA 审核人员

1. **打开映射文档** - 选择对应领域的映射表
2. **查看候选词** - 找到"候选词 (待审核)"部分
3. **填写审核意见** - 在"审核状态"列标记 ✅/⚠️/❌
4. **添加备注** - 在"备注"列说明理由
5. **返回文档** - 保存后提交给开发人员

### 对于开发人员

1. **收集已审核词** - 筛选审核状态为 ✅ 的候选词
2. **更新 JSON** - 添加到 `app/knowledge/intent_keywords.json`
3. **运行测试** - 执行 `test_v46_p0_validation.py`
4. **提交代码** - git commit 并更新版本号

### 对于领域专家

1. **创建新文档** - 复制模板创建新领域映射表
2. **提取关键词** - 从专业文献提取家长常用表达
3. **标注来源** - 记录页码和临床含义
4. **提交审核** - 发送给 BCBA 审核

## 审核流程

```
新词收集 → AI 初步分类 → BCBA 审核 → 开发整合 → 验证测试
   ↓           ↓            ↓          ↓          ↓
文献/社群   映射到假设   标记状态    更新 JSON   运行测试
```

### 审核标准

| 等级 | 标准 | 操作 |
|------|------|------|
| ✅ 通过 | 高频临床表达，有明确依据 | 整合到核心词库 |
| ⚠️ 修改 | 语义模糊或有更优表达 | 修改后整合 |
| ❌ 删除 | 过于主观/生造/重复 | 不整合 |
