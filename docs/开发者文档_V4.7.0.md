## 三、开发指南

### 3.1 添加新关键词

**步骤**:

1. **编辑 `intent_keywords.json`**:
```json
{
  "H_NEW_HYPOTHESIS": [
    "新关键词 1",
    "新关键词 2"
  ]
}
```

2. **更新版本号**:
```json
{
  "version": "V4.7.1",
  "last_updated": "2026-03-14"
}
```

3. **验证测试**:
```bash
python3 test_v464_validation.py
```

4. **Git 提交**:
```bash
git add app/knowledge/intent_keywords.json
git commit -m "V4.7.1: 添加新关键词"
```

### 3.2 添加新假设

**步骤**:

1. **编辑 `hypothesis_network.json`**:
```json
{
  "id": "H_NEW_HYPOTHESIS",
  "name": "新假设名称",
  "description": "描述",
  "prior_probability": 0.30,
  "evidence_rules": {
    "supporting": ["证据 1", "证据 2"],
    "contradicting": ["反对证据"]
  }
}
```

2. **添加鉴别性问题**:
```json
"discriminating_questions": {
  "H_NEW_HYPOTHESIS_vs_H_OTHER": {
    "primary_hypothesis": "H_NEW_HYPOTHESIS",
    "field_mapping": "new_field",
    "question": "鉴别问题"
  }
}
```

3. **添加意图关键词**:
```json
"intent_mapping": {
  "H_NEW_HYPOTHESIS": [
    "关键词 1",
    "关键词 2"
  ]
}
```

## 六、故障排除

### 6.1 常见问题

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 服务无法启动 | 端口占用 | `lsof -i :8000` 检查端口 |
| API 返回 500 | LLM API 密钥无效 | 检查 `config.yaml` |
| 意图识别错误 | 关键词未更新 | 重启服务 |
| 响应慢 | 网络问题 | 检查网络连接 |

### 6.2 日志位置

| 日志 | 位置 |
|------|------|
| **后端日志** | `server.log` |
| **前端日志** | `frontend.log` |
