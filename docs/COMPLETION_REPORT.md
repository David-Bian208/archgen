## 🎯 文档特色

### 1. 语言风格

**家长视角：**
- ✅ 使用"我们""宝贝"等亲切称呼
- ✅ 避免专业术语（如"ABC 分析""功能假设"）
- ✅ 用具体例子代替抽象说明

**示例对比：**

| ❌ 技术语言 | ✅ 家长语言 |
|------------|------------|
| "输入行为描述" | "描述孩子的行为" |
| "系统返回分析报告" | "您会看到一份报告" |
| "干预策略游戏化" | "可以用游戏方式练习" |
| "提交反馈数据" | "帮助我们改进" |

---

### 2. 结构设计

**快速指南：** 3 分钟上手 → 实际使用 → 提交反馈  
**详细手册：** 完整流程 → 技巧说明 → FAQ → 反馈  
**反馈表：** 基本信息 → 评分 → 详细反馈 → 开放建议

**渐进式设计：**
- 新用户：快速指南（3 分钟）
- 深度用户：详细手册（10 分钟）
- 反馈用户：反馈表（5-10 分钟）

---

### 3. 实用导向

**包含大量实际示例：**

✅ 好的行为描述示例：
```
在幼儿园做操时，小明一直盯着旁边的小朋友看，
不跟着做动作。老师提醒后，他看看老师，
又开始看小朋友。
```

✅ 游戏化干预示例：
```
"变化大冒险"游戏：
1. 预告魔法：提前 5-10 分钟用计时器告诉孩子
2. 微小挑战：从超级小的变化开始
3. 选择权力：给孩子两个选择
4. 成功庆祝：热烈庆祝"你做到了！"
```
