## 问题 2："📖 叙事分析"改名 ❌

### 用户反馈
> "📖 叙事分析改个名字，比如详细说明之类的，你帮我想一想"

### 当前名称
```
📖 叙事分析
```

**问题：** "叙事"太学术/文学化，家长可能不理解

---

### 建议新名称

| 选项 | 优点 | 缺点 |
|------|------|------|
| 📖 详细说明 | 用户建议，直白 | 略显平淡 |
| 📖 深度解读 | 体现专业性 | 还是有点学术 |
| 📖 老师怎么看 | 温暖，有对话感 | 可能太长 |
| 📖 专业视角 | 简洁 | "专业"还是有点距离感 |
| 📖 综合分析 | 中性 | 缺乏温度 |
| 📖 行为解读 | 直白易懂 | - |
| 📖 观察与分析 | 完整 | 略长 |

**推荐：** `📖 行为解读` 或 `📖 详细说明`

---

### 修复方案

**文件：** `frontend/App.vue`

**修改位置：** line 101-103 左右

**修改前：**
```vue
<!-- 叙事分析 -->
<div v-if="insightReport.narrative_analysis?.narrative_summary" class="narrative-section">
  <h3>📖 叙事分析</h3>
```

**修改后：**
```vue
<!-- 行为解读 -->
<div v-if="insightReport.narrative_analysis?.narrative_summary" class="narrative-section">
  <h3>📖 行为解读</h3>
```

**注意：** class 名 `narrative-section` 保持不变（只改显示文本）
