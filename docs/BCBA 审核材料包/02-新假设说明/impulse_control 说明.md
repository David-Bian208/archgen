## 一、假设定义

### 1.1 基本信息

| 属性 | 内容 |
|------|------|
| **假设 ID** | impulse_control |
| **名称** | 冲动控制挑战 |
| **描述** | 难以抑制冲动行为或情绪反应，行动先于思考 |
| **先验概率** | 0.30 |
| **层级** | 中层归因假设 |
| **父假设** | H_ESCAPE (逃避) |
| **子假设** | - |

### 1.2 临床定义

冲动控制挑战指个体在特定情境下难以抑制即时反应，表现为：
- 行为冲动：行动先于思考，不考虑后果
- 情绪冲动：情绪反应强烈且快速，难以调节
- 言语冲动：插嘴、抢话、脱口而出

## 二、证据规则

### 2.1 支持证据 (7 条)

| 证据 | 行为表现 | 临床意义 |
|------|----------|----------|
| 冲动 | 突然行动，不考虑后果 | 冲动控制弱 |
| 控制不住自己 | 自我报告无法控制 | 自我意识存在但控制困难 |
| 行动先于思考 | 做事不考虑后果 | 认知 - 行为脱节 |
| 插嘴 | 别人说话时打断 | 言语冲动 |
| 抢东西 | 直接从别人手中拿 | 行为冲动 + 社交技能弱 |
| 不耐烦 | 等待困难 | 延迟满足能力弱 |
| 容易发脾气 | 情绪爆发阈值低 | 情绪冲动 |

### 2.2 反对证据 (2 条)

| 证据 | 行为表现 | 临床意义 |
|------|----------|----------|
| 能等待 | 可以延迟满足 | 冲动控制良好 |
| 三思而后行 | 行动前思考 | 冲动控制良好 |

## 三、鉴别诊断

### 3.1 与相关假设的区别

| 假设 | 核心特征 | 与 impulse_control 的区别 |
|------|----------|--------------------------|
| **emotional_regulation** | 情绪调节困难 | 侧重情绪强度，impulse_control 侧重行为抑制 |
| **H_ATTENTION** | 寻求关注 | 寻求关注是有目的的，冲动是无目的的 |
| **H_ESCAPE_DIFFICULTY** | 逃避难度 | 逃避是有目的的，冲动是即时反应 |

### 3.2 共病情况

| 共病 | 发生率 | 说明 |
|------|--------|------|
| ADHD | 高 | 冲动是 ADHD 核心症状之一 |
| ODD (对立违抗) | 中 | 冲动 + 违抗行为 |
| 情绪障碍 | 中 | 冲动 + 情绪不稳定 |
