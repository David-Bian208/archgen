## 指令一：修正核心分析逻辑

### 关键分析原则

**当行为模式表现为"有提示则执行，无提示则中断/发呆"，且任务具有明确的社会性目标时**：

```
❌ 旧判断：自动强化（自我刺激）
✅ 新判断：在提示依赖下的逃避（任务失败/认知负荷）
```

### 推理链

```
1. 识别模式：这是典型的"视觉提示依赖"行为
   → "看老师就做得好，不看就容易发呆"

2. 功能判断：主要功能并非自我刺激（自动强化），而是为了逃避
   → 逃避因提示中断而导致的"任务执行失败"或"认知过载"
   → 这是一种特殊形式的"逃避"

3. 能力缺口定位：
   → 工作记忆：记不住多步骤序列
   → 持续性注意：无法在提示间隙自我维持任务表征

4. 环境整合：
   → 嘈杂、拥挤等环境因素加剧了对单一、清晰视觉提示的依赖
```

### Prompt 变更

**V3.3 分析 Prompt**：
```python
【关键分析原则】
当行为模式表现为"有提示则执行，无提示则中断/发呆"，且任务具有明确的社会性目标时：
- 首要功能假设应从"自动强化"修正为"在提示依赖下的逃避（任务失败/认知负荷）"
- 这是特殊形式的"逃避"——逃避的是因提示中断而导致的"任务执行失败"或"认知过载"
- 能力缺口：工作记忆和持续性注意的挑战
- 环境整合：嘈杂、拥挤等环境因素会加剧对单一、清晰视觉提示的依赖
```

## 指令二：重写报告的语言与结构

### 从"模块堆砌"到"叙事"

**V3.2 结构**：
```
1. 我们看到了什么
2. 专业视角
3. 这可能意味着什么
4. 我们可以尝试什么
5. 一个小思考
```

**V3.3 结构**：
```
1. 我们看到了什么（精简事实 + 认可家长）
2. 核心洞察：这意味着什么？（金句，直击本质）
3. 专业视角：为什么会这样？（用家长能懂的语言解释提示依赖模型）
4. 我们可以尝试什么：一个具体的起点（超具体的"第一招"）
5. 一个小思考（引导家长成为合作者）
```

---

### 各部分详解

#### 1. 我们看到了什么（精简事实）

**V3.2**：
```
在幼儿园早操时，周围有点吵闹，其他小朋友都在跟着老师做，
孩子排队时站着发呆，眼睛不跟随老师。
```

**V3.3**：
```
在室内跳操时，您敏锐地发现孩子"看老师就做得好，不看就容易发呆"，
这种观察非常精准。
```

**关键变化**：
- 精简事实描述
- 认可家长的观察（"您敏锐地发现"、"这种观察非常精准"）

---

#### 2. 核心洞察：这意味着什么？（金句）

**V3.2**：
```
孩子可能觉得早操有点难跟上，或者不太明白该做什么，
所以用发呆来暂时'躲开'这个让他感到困惑或无聊的活动。
```

**V3.3**：
```
"孩子不是不努力，而是他的'动作记忆'需要靠眼睛实时'充电'。
一旦看不到老师，他的'任务图纸'就模糊了。"

或：

"这更像是'不知道眼睛该看哪里时，手和脚就不知道该干什么'，
而不是'不想做'。"
```

**关键变化**：
- 必须生成"金句"，直击本质
- 用比喻让家长秒懂
- 为行为"正名"（不是不努力，是需要支持）

---

#### 3. 专业视角：为什么会这样？

**V3.2**：
```
这是一种"任务参与度不足行为"。最可能的功能是 escape（逃避），
孩子通过发呆来减少对早操任务的参与。
这可能反映了执行功能方面的挑战，如注意力维持、任务跟随或社交参照能力。
```

**V3.3**：
```
从专业角度看，这属于"提示依赖"。孩子需要持续看着老师来"刷新"
下一步该做什么的记忆。当视线离开，不是他不想做，而是"任务指令"中断了。
这常与工作记忆的发育特点有关。嘈杂的环境（音乐、其他小朋友）
让他更依赖老师这个最明确的提示。
```

**关键变化**：
- 用家长能懂的语言（"刷新记忆"、"任务指令中断"）
- 解释"提示依赖"模型
- 连接环境因素

---

#### 4. 我们可以尝试什么：一个具体的起点

**V3.2**：
```
降低任务复杂度并增加个体化支持，如提供视觉提示或简化指令。
例如，可以在孩子面前贴一个简单的步骤图，或者在他看老师时立即给予微笑鼓励。
```

**V3.3**：
```
基于此，我们可以尝试"预装一个动作"的策略：在跳操前，和他一起夸张地
摆出第一个动作（如"开飞机"），并大声说"记住哦，第一个是开飞机！"。
目的是帮他把一个外部提示（老师动作）转化为一个内部提示（自己脑子里的
口诀或画面）。下次他发呆时，可以轻声提示"开飞机！"，看他能否接上。
```

**关键变化**：
- 超具体的"第一招"
- 与核心洞察直接对应
- 有操作步骤、有原理说明、有验证方法

---

#### 5. 一个小思考（引导家长成为合作者）

**V3.2**：
```
下次在类似情况下，如果我们提前给孩子一个简单的视觉提示
（比如用手指一下要关注的人或物），你猜他会有什么不同的反应吗？
```

**V3.3**：
```
您是否发现，在那些他"不需要看就能做"的活动里（比如唱熟悉的儿歌、
玩熟悉的拼图），任务步骤是不是更简单、或者他已经练习了无数遍？
这或许能帮助我们找到帮他"记住动作"的钥匙。
```

**关键变化**：
- 从"还有哪些类似情况"升级为"启发式思考"
- 引导家长发现规律
- 为后续干预提供线索
