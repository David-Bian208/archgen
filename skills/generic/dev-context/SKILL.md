# dev-context Skill

**通用开发技能 - 项目上下文**

---

## 🎯 用途

帮助开发工程师（trae）理解项目结构、模块关系、API 定义

---

## 🎤 唤醒词（5 个）

### 项目结构
- `查看项目结构` - 显示项目目录结构
- `这个项目有哪些模块` - 列出所有模块

### 模块查询
- `这个模块依赖哪些模块` - 查看模块依赖
- `[模块名] 在哪里` - 查找模块位置

### API 查询
- `API 定义在哪里` - 查看 API 接口定义
- `这个接口的参数是什么` - 查看接口参数

---

📁 项目结构

**目录结构：**

**核心模块：**
