# dev-test Skill

**通用开发技能 - 测试**

---

## 🎯 用途

帮助开发工程师（trae）生成、运行、分析单元测试

---

## 🎤 唤醒词（5 个）

### 测试生成
- `为这个函数生成单元测试` - 生成测试模板
- `为这个模块生成测试` - 为整个模块生成测试

### 测试运行
- `运行测试` - 运行 pytest 测试
- `运行这个测试文件` - 运行指定测试文件

### 测试分析
- `查看测试覆盖率` - 查看覆盖率报告
- `测试为什么失败` - 分析失败原因

---

## 📝 输出格式

### 生成测试

```
🧪 生成单元测试

**文件：** [文件名]
**函数：** [函数名]
**测试框架：** pytest

**生成测试：**
```python
def test_xxx():
    # 测试代码
```

**测试用例：**
1. 正常情况
2. 边界情况
3. 异常情况

**运行命令：** pytest tests/test_xxx.py -v
```

### 运行测试

```
🧪 运行测试

**测试文件：** tests/test_xxx.py
**命令：** pytest tests/test_xxx.py -v

**结果：**
- 通过：X/X
- 失败：0
- 跳过：0

**耗时：** 0.5s
```

### 测试失败分析

```
🧪 测试失败分析

**失败测试：** test_xxx

**错误信息：**
AssertionError: expected 2 but got 3

**失败原因：**
[分析原因]

**修复建议：**
1. [建议 1]
2. [建议 2]
```
