# dev-logs Skill

**通用开发技能 - 日志查看**

---

## 🎯 用途

帮助开发工程师（trae）查看、搜索、分析应用日志

---

## 🎤 唤醒词（5 个）

### 日志查看
- `查看最近日志` - 查看最近的日志
- `查看错误日志` - 只查看错误日志

### 日志搜索
- `搜索包含"[关键词]"的日志` - 搜索日志
- `查看今天的日志` - 按时间过滤

### 日志分析
- `分析这个错误` - 分析错误原因
- `日志里有什么异常` - 异常检测

---

## 📝 输出格式

### 查看日志

```
📋 日志查看

**日志文件：** logs/app.log
**时间范围：** 最近 100 行

**日志内容：**
2026-04-01 10:00:00 INFO - 应用启动
2026-04-01 10:00:01 INFO - 加载配置
2026-04-01 10:00:02 ERROR - 连接失败
...

**统计：**
- INFO: 80
- WARNING: 15
- ERROR: 5
```

### 日志搜索

```
🔍 日志搜索

**关键词：** timeout
**匹配结果：** 5 条

**匹配内容：**
1. 2026-04-01 10:00:02 ERROR - Connection timeout
2. 2026-04-01 10:05:15 ERROR - Request timeout
...
```

### 错误分析

```
🔍 错误分析

**错误类型：** TimeoutError
**出现次数：** 5 次
**首次出现：** 2026-04-01 10:00:02

**可能原因：**
1. 网络连接问题
2. 服务器响应慢
3. 超时设置过短

**修复建议：**
1. 检查网络连接
2. 增加超时时间
3. 优化查询性能
```
