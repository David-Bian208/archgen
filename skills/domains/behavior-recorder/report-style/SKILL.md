# report-style Skill

**领域技能 - 报告风格检查（行为观察伙伴专用）**

---

## 🎯 用途

检查报告是否符合临床语言风格规范

---

## 🎤 唤醒词（4 个）

### 风格检查
- `检查这个报告的风格` - 检查报告语言
- `是否符合临床规范` - 检查规范性

### 规则查询
- `报告风格有哪些要求` - 查看风格规范
- `避免哪些表述` - 查看禁止用语

---

## 📊 检查项

| 检查项 | 正确 | 错误 |
|--------|------|------|
| 视角 | "我们觉得..." | "临床上通常..." |
| 能力描述 | "还没学会" | "不会" |
| 干预用词 | "练习" | "训练" |
| 句子长度 | <30 字 | 长句 |
| 比喻 | 具体行为细节 | "社交雷达"等通用比喻 |

---

## 📝 输出格式

### 风格检查

```
📋 报告风格检查

**检查项：** 5 项
**通过：** X/5

**检查结果：**
✅ 视角：使用"我们"视角
✅ 能力描述：使用"还没学会"
❌ 句子长度：发现 3 个长句（>30 字）
✅ 干预用词：使用"练习"
❌ 比喻：发现通用比喻"社交雷达"

**修改建议：**
1. 缩短第 2 段第 3 句
2. 替换"社交雷达"为具体描述
```
