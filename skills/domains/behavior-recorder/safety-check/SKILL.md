# safety-check Skill

**领域技能 - 安全关键词检测（行为观察伙伴专用）**

---

## 🎯 用途

检测行为描述中的安全关键词，触发安全优先模式

---

## 🎤 唤醒词（4 个）

### 安全检测
- `检查这个行为是否安全` - 安全检测
- `有危险吗` - 快速检测

### 规则查询
- `安全规则有哪些` - 查看安全规则
- `危险关键词列表` - 查看关键词

---

## 📊 安全关键词

### 高优先级（直接触发）

```
爬高、危险、利器、冲撞、跑到马路、触电、溺水、水池、
从高处、跳下、攀爬、窗户、阳台、烫、
尖锐、刀子、剪刀、电源、插座、车祸、交通
```

### 需要上下文确认

```
火 → 需要"玩火/点火/烧"等上下文
     排除"发火/生气/脾气"（发脾气）
```

---

## 📝 输出格式

### 安全检测

```
⚠️ 安全检测

**行为描述：** [描述]
**检测结果：** 危险/安全

**触发关键词：**
- 关键词 1（高优先级）
- 关键词 2（需要上下文）

**建议：**
启用安全优先模式，优先处理安全问题
```
