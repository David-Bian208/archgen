## 场景分类体系

### 1️⃣ 共同游戏/加入游戏场景

**场景特征：**
- 判断自己是否被接纳
- 监测游戏同步性
- 理解动态游戏规则

**典型行为：**
- "以为别人在和她玩，其实别人没有"
- "规则有 10 条，她以为只有 3 条，不停重复"
- "沉浸在自己的游戏脚本里"

**核心能力缺口：**
- 社交信号监测弱
- 认知灵活性不足

**✅ 匹配干预：** "开始信号"类游戏
```
游戏名称：我们开始玩了吗信号练习
核心思路：将抽象的"我们是否在一起玩"转化为具体的身体动作信号
关键步骤：
1. 设定专属连接信号（击掌、对暗号）
2. 家庭演练：游戏前必须完成信号
3. 随时暂停→定格→重启
4. 真实场景提示
```

---

### 2️⃣ 对话/介绍/轮流场景

**场景特征：**
- 单向输出 vs 双向交流
- 任务化执行 vs 动态互动
- 忽略对方反应

**典型行为：**
- "像完成任务一样介绍，没有关注对方有没有在注意她"
- "说完名字就觉得任务完成了"
- "没有在意两个小朋友的反应"

**核心能力缺口：**
- 社交信号监测弱（未监测对方是否在听）
- 观点采择应用困难（未推断对方想法）

**✅ 匹配干预：** "观察 - 反应"类游戏
```
游戏名称：介绍接力赛
核心思路：将"说完话要看对方反应"转化为可练习的具体步骤
关键步骤：
1. 家庭模拟：A 介绍 B 后，必须等待 B 点头/微笑
2. 明确规则：对方有反应后，才能继续下一步
3. 视觉提示：制作"说完→看脸→等反应"提示卡
4. 真实场景：介绍前提醒"说完名字要做什么？"
```

**变体游戏：**
- 对话乒乓球：你说一句→看我→等我说→我说一句
- 表情猜猜猜：说完话后观察对方表情，猜测对方感受
```

---

### 3️⃣ 规则理解/变化场景

**场景特征：**
- 坚持固定规则
- 难以接受变化
- 仪式化行为

**典型行为：**
- "必须走同一条路线"
- "玩具必须按特定顺序排列"
- "不同意就崩溃"

**核心能力缺口：**
- 认知灵活性不足
- 对可预测性的过度需求

**✅ 匹配干预：** "预告 + 小变化"类游戏
```
游戏名称：规则变变变
核心思路：通过预告增加可预测性，通过微小变化培养灵活性
关键步骤：
1. 提前预告：变化前 5-10 分钟告知
2. 微小变化：从极小的变化开始（换一支笔的颜色）
3. 提供选择权：在可接受范围内给孩子选择
4. 成功强化：接受变化时给予具体反馈
```

---

### 4️⃣ 身体边界/触碰场景

**场景特征：**
- 拥抱不知轻重
- 靠得太近
- 触碰他人过度用力

**典型行为：**
- "拥抱太用力，弄哭小朋友"
- "站得离别人太近"
- "拍别人肩膀太重"

**核心能力缺口：**
- 身体边界感不足
- 触觉调节困难

**✅ 匹配干预：** "身体地图"类游戏
```
游戏名称：身体边界小卫士
核心思路：将抽象的"身体边界"转化为可视化的具体概念
关键步骤：
1. 画身体地图：在纸上画出每个人都有自己的"泡泡"
2. 体验距离：用呼啦圈/绳子演示"舒适距离"
3. 角色扮演：练习"轻轻抱"vs"用力抱"的区别
4. 暗号提示：建立"太近了"的非语言提醒信号
```

---

### 5️⃣ 情绪识别/回应场景

**场景特征：**
- 看不出别人不高兴
- 读不懂面部表情
- 无法识别肢体语言

**典型行为：**
- "别人已经皱眉了，她还继续"
- "看不出小朋友不想玩了"
- "忽略别人的拒绝信号"

**核心能力缺口：**
- 情绪识别困难
- 非语言线索解读能力弱

**✅ 匹配干预：** "情绪侦探"类游戏
```
游戏名称：表情大侦探
核心思路：将情绪识别转化为可练习的"侦探游戏"
关键步骤：
1. 表情卡片：制作高兴/生气/难过/惊讶等表情卡片
2. 猜表情游戏：看照片/视频片段，猜测人物感受
3. 情境配对：什么情况下会有什么表情
4. 实时练习：看动画片时暂停，问"他现在感觉如何"
```
