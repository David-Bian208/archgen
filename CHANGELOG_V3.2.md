## 新增功能

### 1. 首轮环境询问

**V3.2 新话术模板**：
```
感谢您这么细致的观察，这非常有帮助。
为了能更准确地理解当时的情况，我想了解一下当时周围的环境怎么样？
比如是有点吵闹还是比较安静，其他小朋友大多在做什么呢？
```

**智能判断**：
- 如果用户首轮输入已包含环境信息（如"吵闹"、"安静"、"其他小朋友"），则直接问情境
- 否则，首轮必须询问环境

---

### 2. 环境因素纳入分析

**专家拆解新增字段**：
```json
{
  "expert_breakdown": {
    "behavior_pattern": "...",
    "functional_hypothesis": "...",
    "capability_gap": "...",
    "contextual_factors": "环境因素（噪音、拥挤、等待等）如何影响行为"
  }
}
```

**分析 Prompt 更新**：
```
【观察信息】
- 环境：{environment}  ← V3.2 新增
- 情境：{antecedent}
- 孩子的表现：{behavior}
- 当时的回应：{consequence}

【你的分析任务】
...
- 环境因素：环境（噪音、拥挤、等待等）如何影响了行为？
```

## 实测案例

### 完整对话流程

```
👤 早操时发呆

💬 您能注意到孩子在早操时的状态，真的很细心。
   感谢您这么细致的观察，这非常有帮助。
   为了能更准确地理解当时的情况，我想了解一下当时周围的环境怎么样？
   比如是有点吵闹还是比较安静，其他小朋友大多在做什么呢？

👤 有点吵闹，其他小朋友都在跟着老师做

💬 您能注意到环境细节，这很有帮助。
   孩子是在什么活动或什么情况下出现这个情况的呢？

👤 幼儿园早操排队时

💬 明白了，在幼儿园早操排队时。
   当时你观察到孩子具体做了什么，或者没做什么，让你比较在意呢？

👤 眼睛不跟随老师，站着发呆

💬 您描述得很具体。
   当孩子那样做的时候，你或周围人（比如老师）第一时间是怎么回应的呢？

👤 老师继续带着其他孩子做操

✨ 根据我们刚才的交流，我生成了一份简要的分析报告...

📊 观察分析

📝 我们看到了什么
在幼儿园早操时，周围有点吵闹，其他小朋友都在跟着老师做，
孩子排队时站着发呆，眼睛不跟随老师。当时，老师继续带着其他孩子做操。

🔬 专业视角
这是一种"任务参与度不足行为"。在嘈杂的集体环境中，孩子可能因感觉信息过载
而选择"关闭"注意力。最可能的功能是 escape（逃避）—— 通过发呆来减少感官输入。
这可能反映了感觉处理和注意力调节方面的挑战。

💡 这可能意味着什么
孩子可能觉得早操环境太吵了，有点应付不过来，所以用发呆来保护自己，
让自己不被那么多声音和动作淹没。

🌱 我们可以尝试什么
降低环境刺激并提供个体化支持，如让孩子站在队伍边缘减少拥挤感，
或使用降噪耳塞减少噪音干扰。

🤔 一个小思考
如果我们尝试让孩子站在一个相对安静、不那么拥挤的位置，
你猜他的注意力会有什么变化吗？
```

## 总结

V3.2 是**从"行为分析"到"情境分析"**的关键升级：

| 维度 | V3.1 | V3.2 |
|------|------|------|
| **分析视角** | 行为 + 能力 | **行为 + 能力 + 环境** |
| **对话起点** | 情境 | **环境** |
| **报告深度** | 个体分析 | **情境化分析** |
| **干预建议** | 通用策略 | **环境调整策略** |

**核心价值**: 理解孩子行为，不仅要看"他做了什么"和"他能做什么"，更要看"他在什么环境中做"。环境感知让分析更有深度，建议更有针对性。
