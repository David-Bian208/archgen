## 🚀 部署指令

### 方式一：上传压缩包（推荐）

```bash
# 1. 上传压缩包到服务器
scp behavior_recorder_v641_deploy.tar.gz user@your-server:/path/to/

# 2. 在服务器上解压
cd /path/to/
tar -xzf behavior_recorder_v641_deploy.tar.gz

# 3. 配置环境变量
cd deploy_v641
cp .env.example .env
nano .env  # 填入实际的 API Key

# 4. 安装依赖
pip3 install -r requirements.txt

# 5. 启动服务
cd chainlit_v62
bash start_chainlit.sh
```

### 方式二：直接 Git 拉取

```bash
# 1. 进入项目目录
cd /path/to/behavior-recorder/

# 2. 拉取最新代码
git pull origin master

# 3. 安装依赖
pip3 install -r requirements.txt

# 4. 清理缓存
find . -type d -name "__pycache__" -exec rm -rf {} +
find . -name "*.pyc" -delete

# 5. 启动服务
cd chainlit_v62
python3 -m chainlit run app.py --port 8005 --host 0.0.0.0
```
