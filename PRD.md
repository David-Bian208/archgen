## 📋 产品概述

**产品名称：** Behavior Recorder Service（行为观察伙伴）  
**定位：** AI 驱动的自闭症儿童行为观察与评估工具  
**理论基础：** ABC 行为分析理论（Antecedent-Behavior-Consequence）

**核心价值：**
- 帮助家长和治疗师记录儿童行为
- AI 生成行为功能假设
- 提供场景化干预建议
- 生成结构化评估报告

**目标用户：**
- 自闭症儿童家长（核心用户）
- 康复机构治疗师
- 特教老师
- 行为分析师

---

## 🎯 目标用户画像

### 主要用户：自闭症儿童家长

**特征：**
- 年龄：25-45 岁
- 教育背景：大专及以上
- 技术能力：中等（会使用智能手机和常见 APP）
- 痛点：
  - 不知道如何记录孩子行为
  - 看不懂专业评估报告
  - 不知道如何干预
  - 缺乏专业指导资源

**需求：**
- 简单易用的记录工具
- 通俗易懂的分析报告
- 可操作的干预建议
- 随时随地的专业支持

## 📦 功能需求

### MVP（必须做）✅

#### 1. 行为记录模块

**功能描述：** 用户输入 ABC 三要素

**输入字段：**
```
- 情境 (Antecedent)：行为发生前的情境
- 行为 (Behavior)：具体可观察的行为
- 结果 (Consequence)：行为发生后的结果
- 时间：行为发生时间
- 地点：行为发生地点
```

**验收标准：**
- [ ] 支持文本输入
- [ ] 支持语音输入（可选）
- [ ] 必填字段验证
- [ ] 本地保存草稿

---

#### 2. AI 分析模块

**功能描述：** AI 生成行为功能假设

**输入：** ABC 记录 + 儿童基本信息  
**输出：** 行为功能假设 + 置信度

**分析维度：**
```
1. 寻求关注 (Attention)
2. 获取物品/活动 (Tangible)
3. 逃避/回避 (Escape)
4. 自我刺激 (Sensory)
5. 安全优先模式 (Safety)
```

**验收标准：**
- [ ] 分析时间 <10 秒
- [ ] 置信度显示
- [ ] 安全关键词触发安全模式
- [ ] 支持重新分析

---

#### 3. 干预建议模块

**功能描述：** 生成场景化干预策略

**输入：** 行为功能假设 + 场景分类  
**输出：** 干预方向 + 具体策略

**场景分类：**
```
1. 共同游戏场景
2. 对话/介绍场景
3. 规则变化场景
4. 身体边界场景
5. 情绪识别场景
```

**干预策略框架：**
```
- 干预重点：X 能力培养
- 可尝试方向：
  1. 日常生活中创造练习机会
  2. 使用视觉提示辅助理解
  3. 及时强化积极行为
- 免责声明：仅供参考，建议咨询专业治疗师
```

**验收标准：**
- [ ] 建议与场景匹配
- [ ] 建议可操作
- [ ] 免责声明显示
- [ ] 避免具体医疗建议

---

#### 4. 报告生成模块

**功能描述：** 生成结构化评估报告

**报告结构：**
```
1. 摘要（100 字内）
2. 核心挑战分析
3. 行为模式解读
4. 能力缺口分析
5. 可尝试的行动方向
6. 免责声明
```

**报告风格：**
- 温暖、专业、易懂
- 能力发展视角（"还没学会"而非"不会"）
- "我们"视角拉近距离
- 短句优先（每句<30 字）

**验收标准：**
- [ ] 报告结构完整
- [ ] 语言通俗易懂
- [ ] 无专业术语堆砌
- [ ] 支持导出 PDF/图片

## 🎨 界面设计要求

### 整体风格

```
- 简洁、温暖、专业
- 避免医疗感，增加亲和力
- 色彩柔和，避免刺激
```

### 色彩方案

```
主色调：温暖蓝色（信任、专业）
强调色：橙色（温暖、活力）
警告色：红色（仅用于免责声明）
背景色：浅灰/白色（简洁）
```

### 布局原则

```
- 移动端优先
- 单列布局
- 大按钮、大字体
- 清晰的操作引导
```

### 关键页面

#### 首页/游戏大厅
```
- 欢迎语
- 快速开始按钮
- 历史记录入口
- 帮助/说明入口
```

#### 记录页面
```
- ABC 三要素输入框
- 语音输入按钮（可选）
- 保存草稿按钮
- 提交分析按钮
```

#### 报告页面
```
- 报告标题
- 各模块内容
- 免责声明（红色框）
- 导出/分享按钮
- 重新分析按钮
```

## ⚠️ 边界与限制

### 功能边界

**本产品不做：**
- ❌ 医学诊断
- ❌ 治疗方案制定
- ❌ 药物建议
- ❌ 替代专业治疗师

**本产品定位：**
- ✅ 辅助工具
- ✅ 参考建议
- ✅ 记录和分析工具
- ✅ 家长教育工具

---

### 法律合规

**免责声明：**
```
本报告由 AI 生成，仅供参考，不构成医学诊断或治疗建议。
请在专业治疗师指导下使用。
```

**隐私保护：**
- 用户数据本地存储（优先）
- 如需云端存储，需用户同意
- 不收集个人身份信息
- 不向第三方共享数据
