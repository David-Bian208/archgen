## 🔧 优化任务（小治）

### 优化方向

#### 1. 简化行为分析流程

**当前流程（4 步）：**
```
ABC 解码 → 模式提炼 → 假设生成 → 证据权衡 → Agent 整合
```

**优化后流程（2 步）：**
```
ABC 场景 → 行为分析 Agent（单步完成）
```

**实现方式：**
- 将 4 个 Skills 合并到 Agent 中
- 单次 LLM 调用完成所有分析
- 减少 4 次 API 调用 → 1 次 API 调用

#### 2. 添加行为分析缓存

**文件：** `app/agents/behavior_analysis/cache.py`

**功能：**
- 缓存相同 ABC 场景的分析结果
- 缓存过期时间：10 分钟
- 减少重复 LLM 调用

#### 3. 优化提示词

**当前提示词：** 分步推理，较长

**优化提示词：** 单次推理，精简
