### 1️⃣ 重构第四部分"计划"为四步结构

**新标题**: 四、我们可以这样开始

**四步逻辑**:
1. **💡 核心思路** - 1-2 句话通俗解释从"问题"到"解决方案"的逻辑
2. **🎮 我们的计划** - 具体方法和生动案例
3. ** 成功的画面** - 清晰、可衡量的成功标准
4. **🚀 第一步行动与观察** - 立即能做的第一步和最简单的观察法

**示例输出**:
```
💡 核心思路
针对孩子"需要看着老师才能记住动作"的情况，我们将帮助他把需要紧盯的"外部提示"，转化为他自己能使用的"内部密语"。就像给他装一个"动作小闹钟"，不需要看老师，自己也能想起来。

🎮 我们的计划：动作密语
我们将一起玩"动作密语"游戏。比如，从体操中选出几个关键动作，把它们变成你们的专属暗号，通过有趣的小游戏反复练习。每次玩的时候，都用这个暗号来提醒孩子，让他在快乐中记住动作。

 成功的画面
在 3 天内，当您说出"动作密语"时，孩子能在 3 秒内做出目标动作，且 10 次尝试中至少成功 8 次。

🚀 第一步行动与观察
今天或明天，找个轻松时刻，玩一次"动作密语"变装游戏。观察他听到指令后，是反应迅速（快），还是需要等待（慢）？简单记在日历上就好。记住，第一次玩不重要，重要的是开始！
```

### 3️⃣ 彻底修复文本乱码

**后端修复** (`app/agents/intervention_planner.py`):
```python
def sanitize_text(self, text: str) -> str:
    """文本安全处理 - 清理或转义特殊字符"""
    text = text.replace("&", "&amp;")
    text = text.replace("<", "&lt;")
    text = text.replace(">", "&gt;")
    text = text.replace('"', "&quot;")
    text = text.replace("'", "&#39;")
    text = text.replace("≥", "&ge;")
    text = text.replace("≤", "&le;")
    # % 保留为纯文本
    return text
```

**前端加固** (`frontend/utils/filters.js`):
```javascript
export function escapeText(text) {
  if (text == null) return ''
  return String(text)
    .replace(/&/g, '&amp;')
    .replace(/</g, '&lt;')
    .replace(/>/g, '&gt;')
    .replace(/"/g, '&quot;')
    .replace(/'/g, '&#039;')
    .replace(/%/g, '％') // 全角百分号
    .replace(/≥/g, '≥') // HTML 实体
    // ... 更多特殊字符处理
}
```

**双重保障**:
- 后端生成时进行文本安全处理
- 前端渲染时使用全局过滤器
- 特殊字符（%, ≥, ", !）在任何位置均无乱码

## 技术实现细节

### 四步结构生成流程

```python
# 1. 提取关键信息
success_criteria = phase.get("success_criteria", "")
strategy_details = phase.get("strategy_details", "")

# 2. 解析成功标准
parsed_success = self._parse_success_criteria(success_criteria)
# → {time_frame: "3 天", behavior: "目标动作", percentage: "80%", response_time: "3 秒"}

# 3. 提取策略关键词
strategy_keyword = self._extract_strategy_keyword(strategy_details)
# → "动作密语" 或 "大鹏展翅"

# 4. 生成四步内容
core_idea = self._generate_core_idea(core_principle, scenario, strategy_keyword)
our_plan = self._generate_our_plan(phase, strategy_keyword, session_context)
success_picture = self._generate_success_picture(parsed_success, strategy_keyword)
first_step = self._generate_first_step(phase, strategy_keyword, session_context)

# 5. 文本安全处理
plan = self._sanitize_plan_text(plan, planner)
```

### 文本安全处理流程

```
用户输入 → LLM 生成 → 后端 sanitize_text() → 前端 $escape() → 安全渲染
                ↓
         特殊字符转义
         (&, <, >, ", ', ≥, ≤)
```
