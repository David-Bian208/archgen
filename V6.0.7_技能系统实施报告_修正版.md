## 🎯 关键修正（小治建议）

### 修正 1：放弃装饰器模式

**原方案（错误）：**
```python
@tracer.instrument
def skill_code_review(arguments): ...
```

**问题：**
- ❌ 技能是 YAML 声明，不是 Python 函数
- ❌ OpenClaw 通过 API 调用，非本地函数执行
- ❌ 装饰器无法注入

**修正方案（正确）：**
```python
# 直接调用追踪器
tracer = SkillTracerPlugin()
tracer.track(skill_id, arguments, latency_ms, status)
```

---

### 修正 2：动态加载追踪器

**原方案（错误）：**
```python
from .claw.plugins.skill_tracer import SkillTracerPlugin  # 相对导入失败
```

**修正方案（正确）：**
```python
# 动态加载
import importlib.util
tracer_path = Path(__file__).parent / "plugins" / "skill_tracer.py"
spec = importlib.util.spec_from_file_location("skill_tracer", tracer_path)
skill_tracer = importlib.util.module_from_spec(spec)
spec.loader.exec_module(skill_tracer)
```
