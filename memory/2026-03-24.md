## 🔧 技术决策

### 为什么选择云服务器而不是内网穿透？

1. **稳定性**：内网穿透免费版限速、不稳定
2. **规模**：100+ 用户测试需要可靠架构
3. **成本**：~150 元/月，可控
4. **复用性**：测试完成后可用于其他项目

### Python 版本兼容

- 服务器默认 Python 3.6.8
- FastAPI 0.104+ 需要 Python 3.8+
- **解决方案**：使用 FastAPI 0.68.0 + uvicorn 0.15.0（Python 3.6 兼容版）
