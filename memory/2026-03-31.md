# 2026-03-31 - 行为观察助手故障诊断与修复

## 核心问题

### P0: 前端"思考后无回复"问题
**用户反馈：** "提问后显示思考中，然后跳出对话框图案，没有任何答复"

**根因诊断（战舰 🛳️）：**
1. **主要 Bug**: `frontend/App.vue` 中 `scrollToBottom()` 在 `in_progress` 状态下永远不会被调用
   ```javascript
   // ❌ 原代码
   } else if (data.status === 'in_progress') {
     this.loading = false
     return  // 直接 return，下面的 scrollToBottom() 不执行
   }
   this.scrollToBottom()  // 永远到不了这里
   ```
2. **次要 Bug**: App.vue 第 552 行 JavaScript 语法错误（重复代码）导致 Vite 编译失败 → 空白页

**修复方案：**
1. 在 `in_progress` 分支内添加 `scrollToBottom()` 调用
2. 删除重复的 `inOrderedList = false` 代码行

**关键教训：**
- 文本 > 大脑 📝 - 所有 bug 必须文档化
- 前端修改后必须清除浏览器缓存（Ctrl+Shift+R）
- Vite 热重载有时会失效，需要重启服务
