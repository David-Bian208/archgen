# 2026-03-13 行为记录器系统修复日志

## 重要修复：V4.5.18

### 问题发现（真人测试反馈）

1. **重复提问问题** - 同一问题问 2 次
2. **"不知道"未处理** - 用户回答"不知道"被视为无效，导致重复提问
3. **ABC 逻辑缺失** - 模糊输入（如"眼神关注有问题"）直接问 C，缺少 A 和 B
4. **意图关键词缺失** - H_AUTOMATIC 相关关键词完全缺失（如"转圈"、"摇晃"等）

### 根因分析

1. **重复提问**: 3 个函数的 `asked_fields` 阈值不一致
   - `decide_next_question`: >= 1 ✅
   - `_find_best_discriminating_question`: >= 2 ❌
   - `_find_hypothesis_specific_question`: >= 2 ❌

2. **"不知道"处理**: 规则提取未处理"不知道"关键词

3. **ABC 逻辑**: 首轮模糊输入未记录为行为描述

4. **意图映射缺失**: INTENT_MAPPING 中完全没有 H_AUTOMATIC 相关关键词

### 修复内容

#### 修复 1: 统一重复提问阈值

**文件**: `app/agents/clinical_reasoning_engine.py`

```python
# 修改前
if asked_fields.get(field_mapping, 0) >= 2:

# 修改后
if asked_fields.get(field_mapping, 0) >= 1:  # 只允许问 1 次
```

**影响函数**:
- `_find_best_discriminating_question` (第 670 行)
- `_find_hypothesis_specific_question` - 第一轮 (第 690 行)
- `_find_hypothesis_specific_question` - 第二轮 (第 725 行)

#### 修复 2: "不知道"处理

**文件**: `app/agents/structured_assessor_v4.py`

```python
# 新增代码
if user_input in ["不知道", "不清楚", "没注意", "忘了", "说不清", "不确定"]:
    extracted["immediate_consequence"] = "观察者未注意/无法确定"
```

#### 修复 3: ABC 信息收集 fallback

**文件**: `app/agents/structured_assessor_v4.py`

```python
# 首轮模糊输入记录为行为描述
if state.turn_count == 1 and not extracted.get("antecedent"):
    extracted["behavior_detail"] = user_input
```

### 经验教训

1. **测试覆盖率不足** - 自我刺激行为是常见行为，但关键词完全缺失
2. **阈值一致性** - 多个函数检查同一条件时，阈值必须统一
3. **用户回答有效性** - "不知道"是有效临床信息，应记录而非忽略
4. **ABC 顺序重要性** - 模糊输入应先记录为 B，再追问 A
