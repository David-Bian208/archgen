**方法**: 完全重写 `showAIPlanForm()` 函数

**修复要点**:
1. **结构简化**: 将6层嵌套减少到更合理的结构
2. **正确的try-catch位置**: 
   - 外层try-catch处理模态显示错误
   - 内层try-catch处理表单提交和API调用错误
3. **明确的函数边界**: 确保每个函数都有正确的开头和结尾
4. **错误处理改进**: 添加finally块确保按钮状态恢复

**关键修复代码**:
```javascript
submitBtn.onclick = async () => {

    } finally {
        // 确保按钮状态恢复

        if (currentSubmitBtn) {
            currentSubmitBtn.disabled = false;
            currentSubmitBtn.innerHTML = '<i class="fas fa-check"></i> 生成AI计划';

    }
};
```
